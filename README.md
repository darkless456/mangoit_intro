# 芒果IT公众号说明

## 文字描述

1. 扫描专属二维码
2. 用户注册（是工程师需要工程师注册）
3. **呼叫IT（呼叫本人自动回复、负责人收到呼叫提醒、后台收到报障提醒）**
4. 确认呼叫（呼叫本人和负责人收到确认通知）
5. **委派任务（负责人收到委派通知、工程师收到委派通知）**
6. **工程师提交派单（负责人收到确认通知、工程师收到待确认通知）**
7. **负责人确认派单（工程师收到完成通知，可以离场）**
